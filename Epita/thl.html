Classification des grammaires

Hiérarchie de Chomsky sur les grammaires en fonctions des caractéristiques de ces dernières

(NB: Pour les exemples suivants, chaque lettre de l’alphabet Français représente des éléments non terminaux et inversement pour les lettres grecs)

Grammaire Sentence Monotone :

α→β \alpha → \beta α→β avec ∣α∣≤∣β∣ |\alpha| \le |\beta|∣α∣≤∣β∣

Grammaire Context-Sensitive :

αAβ→αγβ(γ≠ϵ) \alpha A\beta → \alpha \gamma\beta \\ (\gamma ≠ \epsilon) αAβ→αγβ(γ=ϵ) Toute règle comprend un non-terminal entouré de deux mots qui décrivent le contexte dans lequel la variable peut être remplacée.

Grammaire Hors-Context :

A→α A → \alpha A→α où A est un symbole non terminal et α est une chaîne composée de terminaux et/ou de non-terminaux. En clair, la partie gauche est composé que d’un élément.

Grammaire Linéaire Gauche/Droite :

Droite :
A→αBB→βCC→γ A → \alpha B \\ B → \beta C \\ C → \gamma A→αBB→βCC→γ Ainsi tous les éléments non terminaux sont du coté droit (ou respectivement gauche).

Classification

Ainsi Chomsky classe les grammaires comme cela :

  • TYPE 0 → Aucun contrainte
  • TYPE 1 → Monotone / Context-Sensitive
  • TYPE 2 → Hors Contexte
  • TYPE 3 → Linéaire
