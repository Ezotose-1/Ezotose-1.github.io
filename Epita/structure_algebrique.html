Les structures algébriques

Introduction aux STA

(Partie pas à connaître mais pratique pour comprendre la suite )
Avant de parler d’anneau, parlons de groupe.
Un groupe est un ensemble E muni d’une opération telle que :
E×E→E(x,y)→x∗y E × E → E \\ (x, y) → x * y E×E→E(x,y)→x∗y Définition très large qui peut englober beaucoup d’exemples.

Cette opération peut contenir un élément neutre, c’est à dire :
∃ n∈E,∀ x∈En∗x=x∗n=x \exist \ n \in E, \forall \ x \in E \\ n * x = x * n = x ∃ n∈E,∀ x∈En∗x=x∗n=x
Exemples :

  • x + 0 = 0 + x = x ⇒ 0 est élément neutre pour l’addition
  • x - 0 ≠ 0 - x ≠ x ⇒ 0 n’est pas un élément neutre pour la soustraction

L’opération peut contenir un élément symétrique de x, c’est à dire :
∀ x∈E,∃ x′∈Ex∗x′=x′∗x=n \forall \ x \in E, \exist \ x' \in E \\ x * x' = x' * x = n ∀ x∈E,∃ x′∈Ex∗x′=x′∗x=n
Exemples :
0 est élément neutre de ‘+’, ainsi
x + (-x) = (-x) + x = 0 ⇒ -x est élément symétrique pour l’addition

Enfin la loi de composition interne doit être associative, c’est à dire
∀(a,b,c)∈E3a+(b+c)=(a+b)+c \forall (a, b, c) \in E^3\\ a + (b + c) = (a + b) + c ∀(a,b,c)∈E3a+(b+c)=(a+b)+c

Notion d’anneau

Semblable au groupe, un anneau est un ensemble E composé de deux opérations. C’est don un triplet (A, +, ×) composé de

  • Un ensemble A avec les deux éléments neutre
  • Deux opérations binaires allant de A vers A tel quel
    A+A→AA×A→A A + A → A \\ A × A → A A+A→AA×A→A appelé addition et multiplication.
  • Des axiomes qui satisfont l’addition :
    • associativité : a + (b + c) = (a + b) + c
    • commutativité : a + b = b + a
    • élément neutre : a + 0A = a + 0A = a
    • inversibilité : a + (-a) = (-a) + a
  • Des axiomes pour la multiplication :
    • associativité : a × (b × c) = (a × b) × c
    • distributivité sur + : a × (b + c) = a × b + a × c
    • élément neutre : a × 1A = 1A × a = a

Un anneau est commutatif s’il satisfait pour tout a, b de A
a×b=b×a a × b = b × a a×b=b×a

Exemples d’anneaux :

Classiquement (Z, +, ×) ou encore R, C, Q avec × le produit * et + l’addition classique. Les polynômes ou fonctions numériques. Les matrices.
On trouve souvent des anneaux sous la forme E/nE ⇒ tous les éléments de {0, …, n-1} modulo n.
Par exemple (Z/5Z, +, ×) est l’anneau composé de {0, 1, 2, 3, 4} et avec :
2+4=(2+4)% 5=1. 2 + 4 = (2 + 4) \%\ 5 = 1. 2+4=(2+4)% 5=1.

Sous-anneaux :

Si (A, +, ×) est un anneau. On dit que B est un sous anneau de A ssi :

  • (B, +) est associatif et admet un élément neutre
  • 1A appartient à B
  • B est stable par ×

(ça revient à dire que B⊂A et B est un anneau )

Notion de corps

Un corps est un anneau pour lequel les éléments sont inversible par × sauf 0A.
Ainsi K est un corps ssi :

  • (K +, ×) est un anneau
  • et (∀x∈K, x ≠ 0A), x × x’ = x’ × x = 1A

Notion d’intégrité

On définit A un anneau intègre s’il est commutatif et si toute relation du type ab = 0 pour deux éléments de A, implique a = 0 ou b = 0.
∀(a,b)∈A²,(a×b=0)⇒(a=0 ou b=0) \forall (a, b) \in A², \\ (a × b = 0) ⇒ (a = 0 \ ou \ b = 0 ) ∀(a,b)∈A²,(a×b=0)⇒(a=0 ou b=0)

Nombre premiers dans les anneaux

Un élément a d’un anneau A est inversible s’il est b∈A tel que
ab=1=ba ab = 1 = baab=1=ba

Pour les nombre premiers dans Z :
si a divise p premier ⇒ (a = 1) ou (a = -1) ou (a = p) ou (a = -p).
⇒ Les nombres premiers sont les éléments inversible dans Z.

Stathme

On appelle stathme sur (A, +, ×) toute application v: A* → N tel que :

  • ∀ a, b∈A, v(a) ≤ v(ab)
  • ∀ a∈A, ∀ b∈A*, ∃q, r∈A, tel que a = bq + r avec r = 0
    ou v® < v(b)

Le stathme sur Z c’est la valeur absolue.
