Arithmétique

Faire de l’arithmétique ⇒ étudier les relations de divisibilité entre les entiers

Rêgles particulières : 1 divise tous les entiers, 0 est divisable par tous les entiers, les nombres premiers ne se divise qu'eux même, etc…

Definition division sur Z :

On dit que k (entier) divise n (entier) s’il existe l (entier) tel que :
m=k×l m = k × l m=k×l ⇒ La notion de divisibilité est relié à une multiplication.

Une division est une relation d’ordre :
un élément peut se diviser lui même car :
m=m×ll=1 m = m × l \\ l = 1 m=m×ll=1

Notion d’anneau

Un anneau est un triplet (A, +, ×) (× ou •) composé de

  • Un ensemble A composé de deux éléments neutre : 0 pour + et 1 pour ×
  • Deux opérations binaires allant de A vers A tel quel
    A+A→AA×A→A A + A → A \\ A × A → A A+A→AA×A→A appelé addition et multiplication.
  • Des axiomes qui satisfont l’addition :
    • associativité : a + (b + c) = (a + b) + c
    • élément neutre : a + 0 = a
    • inversibilité : a + (-a) = (-a) + a
    • commutativité : a + b = b + a
  • Des axiomes pour la multiplication :
    • associativité : a × (b × c) = (a × b) × c
    • élément neutre : a × 1 = a
  • Distributivité gauche : a × (b + c) = a × b + a × c
  • Distributivité droite : (b + c) × a = b × a + c × a

Un anneau est commutatif s’il satisfait pour tout a, b de A
a×b=b×a a × b = b × a a×b=b×a

Exemples d’anneaux :

Classiquement Z, R, C avec × le produit * et + l’addition classique. Les polynômes ou fonctions numériques. Les matrices.

Notion d’intégrité

On définit A un anneau intègre s’il est commutatif et si toute relation du type ab = 0 pour deux éléments de A, implique a = 0 ou b = 0.
∀(a,b)∈A²,(a×b=0)⇒(a=0 ou b=0) \forall (a, b) \in A², \\ (a × b = 0) ⇒ (a = 0 \ ou \ b = 0 ) ∀(a,b)∈A²,(a×b=0)⇒(a=0 ou b=0)

Nombre premiers dans les anneaux

Un élément a d’un anneau A est inversible s’il est b∈A tel que
ab=1=ba ab = 1 = baab=1=ba

Pour les nombre premiers dans Z :
si a divise p premier ⇒ (a = 1) ou (a = -1) ou (a = p) ou (a = -p).
⇒ Les nombres premiers sont les éléments inversible dans Z.

Stathme

On appelle stathme sur (A, +, ×) toute application v: A* → N tel que :

  • ∀ a, b∈A, v(a) ≤ v(ab)
  • ∀ a∈A, ∀ b∈A*, ∃q, r∈A, tel que a = bq + r avec r = 0
    ou v® < v(b)

Le stathme sur Z c’est la valeur absolue.
