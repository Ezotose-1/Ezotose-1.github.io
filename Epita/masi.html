1. Définitions des Signaux

Un signal est forcément continue ou possède un nombre fini/dénombrable de discontinuités de hauteurs finies. Et ce dernier est de module barré :
∣x(t)∣<∞ | x(t) | < \infty ∣x(t)∣<∞
On les a sous la forme de l’équation suivante :
x:I⊂R→Rt→x(t) x: I\subset R → R \\ t → x( t ) x:I⊂R→Rt→x(t)

2. Décalage d’un signal

On manipulera des signaux avec comme valeur de référence le temps t et un possible décalage τ (tau) tel que :
Deˊcalage:x(t−τ) Décalage : x(t - τ) Deˊcalage:x(t−τ)
On a alors un décalage du signal en fonction d’une valeur τ tel que

  • Si τ > 0 : La valeur x(t - τ) est retardé de τ.
    On va chercher un point du signal qui est τ en arrière par rapport à t
  • Si τ < 0 : La valeur x(t - τ) est avancé de τ.
    On va chercher un point du signal qui est τ en avant par rapport à t

3. Corrélation

I. Définition

Est utilisé pour trouver une similarité entre deux signaux.
Avant de voir la corrélation de deux signaux entre eux voyons la corrélation d’un signal avec lui même, c’est à dire l’autocorrélation.
Cela se définit par la superposition d’un signal x avec x retardé de τ.
Fonction d’autocorrélation :
Γxx:τ→<x(t),x(t−τ)> =∫x(t)x(t−τ)‾dt \Gamma xx : τ → <x(t), x(t-τ)> \ = \int x(t) \overline{x(t-τ)}dt Γxx:τ→<x(t),x(t−τ)> =∫x(t)x(t−τ)​dt Γ(τ) = similarité (produit scalaire) entre x et x translaté de τ. Et ceux pour τ prenant toutes les valeurs sur lesquels le signal est définit.
Ainsi là ou la fonction de corrélation est maximale est la position τ pour laquelle les deux signaux ont une ressemblance maximale.

II. Propriétés

  • Γxx(0) = < x(t), x(t) > = ∫|x(t)|² = ||x||²
  • Γxx pair
  • Γxy ≠ Γyx
  • Pour un signal avec lui-même, le max est atteint en 0 ⇒ ressemblance sans décalage.
  • Le bruit n’a pas d’impact sur la corrélation

Pour un signal T-périodique, Γxx est T-périodique. On retiens que
Γxx(τ)=<x(t),x(t−τ)> =∫x(t)x(t−τ)‾dt \Gamma xx (τ) = <x(t), x(t-τ)> \ = \int x(t) \overline{x(t-τ)}dt Γxx(τ)=<x(t),x(t−τ)> =∫x(t)x(t−τ)​dt dans l’espace correcte de x.

Fonction d’intercorrélation :
Similairement à l’auto-corrélation, l’inter-corrélation est la corrélation d’un signal x et y avec y retardé de τ.
Γxy(τ)=<x(t),y(t−τ)> =∫x(t)y(t−τ)‾dt \Gamma xy (τ) = <x(t), y(t-τ)> \ = \int x(t) \overline{y(t-τ)}dt Γxy(τ)=<x(t),y(t−τ)> =∫x(t)y(t−τ)​dt

3. Convolution

I. Définition

Le convolution (ou produit de convolution) est très utilisé en mathématique des signaux mais aussi dans les réseaux de neurones.
Similaire à la corrélation, on définit le produit de convolution d’un signal x et d’un signal y par :
x∗y:R→Rτ→∫Rx(t)y(τ−t)dt x * y : R → R \\ τ → \int_{R}x(t)y(τ-t)dt x∗y:R→Rτ→∫R​x(t)y(τ−t)dt Graphiquement cela correspond au signal de la somme de deux signaux x par un signal y dans le sens inverse que celui de la corrélation :

  • On prend x et y deux signaux
  • On transforme y en y : t → y(-t) ⇒ symétrique par rapport à Oy
  • On décale y vers la gauche d’un facteur τ : y(-t) → y(-t + τ) = y(τ - t)
  • On calcul l’air du produit entre x et y décalé.
  • Lorsque T varie on “glisse” y(τ - t) sur Ox.

II. Propriété :

  • (x * y) = (y * x)
